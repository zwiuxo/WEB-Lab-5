**⚙️ Как запустить проект**
1. Склонировать репозиторий и перейти в папку:
```bash
git clone https://github.com
cd WEB-Lab-5
```
2. Запустить контейнеры (Docker соберет образ PHP с поддержкой MySQL):
```bash
docker-compose up -d --build
```
3. Ссылки:
- **Сайт:** [http://localhost:8080](http://localhost:8080)
- **Управление БД (Adminer):** [http://localhost:8081](http://localhost:8081)
- **База данных:** lab5_db (user: lab5_user, pass: lab5_pass)

**📂 Содержимое проекта**
- `Dockerfile` — сборка PHP с расширением `pdo_mysql`.
- `docker-compose.yml` — описание всех сервисов (Nginx, PHP, MySQL, Adminer).
- `www/db.php` — подключение к БД и автоматическое создание таблицы.
- `www/TaxiOrder.php` — класс с методами для добавления, получения и подсчета заказов.
- `www/process.php` — обработка формы и запись в базу.
- `www/index.php` — главная страница с выводом таблицы и фильтрами.
- `www/form.html` — форма заказа такси.

** Штрафное задание **
- **created_at:** В таблицу добавлено поле даты и времени, сохраняется автоматически.
- **Сортировка:** Записи выводятся в порядке `DESC` (сначала самые новые).
- **Фильтр:** Реализована фильтрация через SQL (показ групп в которых более 2 пассажиров).
- **Статистика:** На главной странице выводится общее количество заказов.
